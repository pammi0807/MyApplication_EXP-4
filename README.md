# MyApplication_EXP-4

Aim

To develop an Android application that adds two numbers entered by the user and displays the result.

Algorithm

1.Start Android Studio and create a new project.

2.Design the layout using TextView, EditText, and Button.

3.Provide two EditText fields to enter numbers.

4.Add a button for performing addition.

5.Add another button for reset.

6.Initialize the UI components in MainActivity.

7.Read values entered by the user.

8.Convert the input strings into numbers.

9.Perform addition of the two numbers.

10.Display the result using TextView.

11.Reset fields when the reset button is clicked.

12.Run the application and verify the result.
